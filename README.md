# PowerBI Sales Analysis Dashboard

## Project Overview
This project analyzes sales data using Power BI to identify trends across
regions, categories, and years.

## Tools Used
- Power BI
- Excel (Dataset)
- GitHub

## Key Insights
- Sales are highest in the East region
- Technology category contributes maximum sales
- Sales show a steady increase over years
